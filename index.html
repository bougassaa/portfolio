Ping Pong Management
Une gestion complète pour l'entreprise de pièces de Ping Pong

Ping Pong Management est une application d'entreprise que j'ai développée pour gérer efficacement toutes les opérations d'une entreprise de fabrication de pièces de Ping Pong. Cette solution polyvalente permet de suivre les pièces tout au long du processus de production, de gérer les bons de commande et de générer des devis pour les clients.

Techniquement, j'ai utilisé PHP Symfony, Doctrine, React et React Native pour créer une application web et mobile moderne. Le backend repose sur une API Restful, offrant une communication fluide entre les différents modules. Le frontend offre une interface réactive et intuitive, tandis que l'application mobile garantit une compatibilité multiplateforme.

Ce projet m'a permis d'exploiter mes compétences en développement web et mobile, en utilisant des technologies de pointe. J'ai pu approfondir ma compréhension des bonnes pratiques de gestion d'entreprise et de conception d'interfaces utilisateur. Je suis fier d'avoir contribué à la création d'une solution complète qui optimise la gestion des opérations au sein de l'entreprise de fabrication de pièces de Ping Pong.
